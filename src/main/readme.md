# Main Charts

## Install

```shell
helm repo add main https://charts.imaginekube.com/main
```

## Contribute

Place your charts under this directory, one chart per separate directory, e.g.

```shell
src/
├── main/
│   ├── example-chart/
│   │   ├── Chart.yaml
│   │   ├── values.yaml
│   │   ├── templates/
│   │   └── ...
│   └── example-chart-2/
│       ├── Chart.yaml
│       ├── values.yaml
│       ├── templates/
│       └── ...
```
