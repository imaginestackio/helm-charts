# Main Charts

## Install

```shell
helm repo add stable https://charts.imaginekube.com/stable
```

## Contribute

Place your charts under this directory, one chart per separate directory, e.g.

```shell
src/
├── stable/
│   ├── example-chart/
│   │   ├── Chart.yaml
│   │   ├── values.yaml
│   │   ├── templates/
│   │   └── ...
│   └── example-chart-2/
│       ├── Chart.yaml
│       ├── values.yaml
│       ├── templates/
│       └── ...
```
